Placemint AI 🚀

Placemint AI is a smart placement preparation platform designed to help students improve their interview readiness, resume quality, and overall confidence during campus placements.

Built with a modern and responsive interface, the platform focuses on making placement preparation simple, interactive, and accessible for students.

✨ Features
AI-powered Resume Analyzer
Mock Interview Practice
Personalized Dashboard
Clean and Responsive UI
Student-focused Experience
Fast and Modern Web Performance
🛠️ Tech Stack
Next.js
React
TypeScript
Tailwind CSS
ShadCN UI
📌 Purpose

The goal of Placemint AI is to support students during placement season by providing tools that improve preparation, confidence, and performance in interviews.

🚀 Getting Started

Clone the repository and install dependencies:

npm install

Run the development server:

npm run dev

Open http://localhost:3000 in your browser.
